# NOTES FOR AWS DEPLOYMENT:
# ==========================================
# 1. For AWS Lambda: Create a deployment package with these dependencies
#    - Use Amazon Linux 2 or AL2023 compatible wheels
#    - Consider using AWS Lambda Layers for large dependencies
#
# 2. For AWS EC2 / ECS:
#    - Install system dependencies first:
#      * sudo yum install -y python3-devel gcc cairo-devel pango-devel gdk-pixbuf2-devel
#      * For WeasyPrint: libffi-devel
#
# 3. For AWS Elastic Beanstalk:
#    - Ensure .ebextensions/ folder contains required system package configs
#    - May need to specify platform-specific packages
#
# 4. OpenSearch:
#    - If using AWS OpenSearch Service, ensure security groups allow access
#    - Use IAM authentication for production
#
# 5. Environment Variables Required:
#    - OPENSEARCH_URL
#    - OPENSEARCH_AUTH (username:password)
#    - OPENAI_API_KEY
#    - TELEGRAM_API_ID
#    - TELEGRAM_API_HASH
#    - AWS_ACCESS_KEY_ID (if not using IAM roles)
#    - AWS_SECRET_ACCESS_KEY (if not using IAM roles)
#    - AWS_REGION
#
# 6. Recommended AWS Services:
#    - AWS OpenSearch Service (for search/analytics)
#    - AWS Lambda (for scheduled tasks)
#    - Amazon ECS/Fargate (for containerized deployment)
#    - AWS RDS (if using PostgreSQL for user management)
#    - Amazon S3 (for file storage)
#    - AWS CloudWatch (for logging)
#    - AWS Secrets Manager (for sensitive credentials)
